1. Which lifecycle method is used to clear the DOM when a component is deleted from the DOM?
    Answer: In class components, it's componentWillUnmount(). In functional components, it's achieved with useEffect().

    2. What is the difference between state and props?
    Answer:
    
    Props:
    Data is passed from a parent component to a child component.
    Data cannot be modified by the child component.
    It is controlled by the parent component.
    State:
    Data of the component can be updated locally.
    Data can be updated using the setState() method.
    It is controlled by the component in which it is declared.

    3. Explain about Router.
    Answer:
    
    Router is a third-party library that can be installed using npm install react-router-dom.
    It is used in a single-page application with multiple views.
    It is used to navigate from one page to another.
    It maps specific components to different URLs, enabling the rendering of distinct UIs for each route.
